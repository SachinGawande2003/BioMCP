# Support

Heuris-BioMCP support is handled through the repository support channels below.

## Primary Support Channel

- Open a GitHub issue: https://github.com/SachinGawande2003/Heuris-BioMCP/issues

## Security Issues

- Follow the process in [SECURITY.md](SECURITY.md) instead of opening a public issue.

## What To Include

- The tool name and exact input that failed.
- Whether the server was local, Render-hosted, or another deployment target.
- Relevant timestamps, deployment URL, and any error payloads or request IDs.
- Whether optional credentials such as `NCBI_API_KEY`, `BIOGRID_API_KEY`, or NVIDIA keys were configured.

## Response Expectations

- Best-effort triage for public issues.
- Security reports follow the response targets documented in [SECURITY.md](SECURITY.md).
